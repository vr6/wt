Contributors to this site's content

Data on this page is made possible through the valuable contributions from several cool folks who helped in different ways. They are helping in fetching data from USCIS by running a script to pull status for a small number of cases. This data is stitched together and visualized on the front page.

If you are interested in maintaining data for any series (one or more) please drop an email to worktheme.dev@gmail.com. You will get a token that is needed for you to update the data on this site. All that you will need to do is running a script for couple of minutes once a day. You can read the script to understand what it is doing. Here it is for download.
